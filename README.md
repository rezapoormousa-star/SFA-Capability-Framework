# Replication Code for "From Efficiency to Quality: A Framework for Linking SFA and Operational Capability"

**Author:** Reza Pourmousa  
**Repository:** SFA-Capability-Framework  

## Description
This repository contains all R scripts required to reproduce the numerical results presented in the paper.  
The code is written in a human‑readable style and can be executed on any standard R environment.

## Requirements
- R version ≥ 4.0  
- Required packages: `frontier`, `dplyr`, `boot`, `parallel`

## Installation
```r
install.packages(c("frontier", "dplyr", "boot", "parallel"))
